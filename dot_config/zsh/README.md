# zsh

I use the [zsh](https://www.zsh.org) shell.

## Config

The shell configuration I use lives in a couple places:
1. `~/.zshrc`: The main workhorse of the configuration. It's commented!
2. `add_path.zsh`: a utility script used in `zshrc` to add paths to $PATH only once
3. `plugins.zsh`: a utility script that functions as a plugin manager.
4. `README.md`: You're reading it.
5. `ssh.zsh`: A helper script for ssh sessions.
