# bin

This folder includes custom scripts I have made.

- `executable_color-picker.sh`: A script for picking a color from the screen, for use in a Wayland session.
   Requires [`slurp`](https://github.com/emersion/slurp), [`grim`](https://sr.ht/~emersion/grim/), and [`convert` (imagemagick)](https://imagemagick.org/script/convert.php).
- `executable_lock.sh` : A script that sets a blurred lockscreen when locked.
   Requires `grim`, `swaylock` and `imagemagick`.
- `executable_update.sh`: A script that attempts to do a system-wide update for flatpaks and distroboxes.
   Requires `flatpak` and `distrobox`.
- `README.md`: You're reading it.
